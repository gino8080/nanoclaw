Guida completa Siri Shortcuts per NanoClaw Telegram

❌

Perché il messaggio appariva scritto da Jarvis?

L'approccio precedente usava sendMessage della Bot API di Telegram. Questo invia un messaggio DAL bot A te — è come se Jarvis scrivesse a sé stesso. NanoClaw non lo vede mai come una domanda dell'utente.

Per far funzionare Jarvis, devi essere TU a inviare il messaggio al bot (come fai normalmente su Telegram). Solo così NanoClaw lo intercetta, processa e risponde.

✅

La soluzione corretta

Usare lo URL scheme di Telegram (tg://) che apre la tua chat con Jarvis con il testo già pronto. Tu tocchi solo Invia. Jarvis risponde normalmente via notifica Telegram.

📋 Passi Shortcut A — Dettagliati

1

Crea un nuovo Shortcut

Apri l'app Shortcuts → tocca + in alto a destra → tocca "Add Action" nella barra in basso.

2

Cerca e aggiungi "Dictate Text"

Nella barra di ricerca scrivi Dictate → seleziona Dictate Text (icona microfono viola).
Nell'azione che appare configura:

  • Language: seleziona Italian dal menu a tendina
  • Stop Listening: seleziona After Pause
3

Aggiungi "URL"

Tocca + sotto l'azione precedente → cerca URL → seleziona l'azione URL (non "Open URLs", quella dopo).

Nel campo testo incolla:

tg://resolve?domain=@TUO_BOT&text=[Dictated Text]
Come inserire [Dictated Text]: dopo aver incollato tg://resolve?domain=@TUO_BOT&text=, tocca l'area testo e poi il pulsante delle variabili (icona a forma di quadratino con freccia, in basso a destra della tastiera). Seleziona "Dictated Text" dalla lista — apparirà come una pillola blu nel campo URL.
4

Aggiungi "Open URLs"

Tocca + → cerca Open URLs → selezionalo.
Nel campo URL dell'azione seleziona la magic variable "URL" (la variabile prodotta dall'azione precedente, apparirà come pillola arancione).

5

Assegna nome e frase Siri

Tocca il nome dello shortcut in alto → rinomina in "Jarvis".
Poi tocca l'icona ⋮ (tre punti) in alto a destra → Add to Siri → registra la frase:
"Hey Siri, Jarvis" oppure "Chiedi a Jarvis".

«Hey Siri, Jarvis»
poi parla → Telegram si apre con il testo pronto → tocchi Invia
🚀 Opzione B — Hands-free con risposta vocale

Come funziona

Aggiungiamo un piccolo server HTTP a NanoClaw che gira sul tuo Mac. Il tuo iPhone (sulla stessa WiFi) chiama questo server, che inserisce il messaggio direttamente nella pipeline e restituisce la risposta. Lo shortcut la legge ad alta voce.

⚙️ Passo 1 — Trova l'IP del tuo Mac

1

Apri Terminale sul Mac

Vai su Applicazioni → Utility → Terminale e lancia il comando:

ipconfig getifaddr en0

Otterrai qualcosa come 192.168.1.15. Questo è il tuo MAC_IP da usare nello shortcut.

2

Verifica che iPhone e Mac siano sulla stessa WiFi

Entrambi devono essere connessi alla stessa rete domestica. Se l'iPhone è su dati mobili non funzionerà (a meno di aggiungere un tunnel pubblico con Cloudflare o ngrok — configurazione avanzata).

⚙️ Passo 2 — Aggiungi il server HTTP a NanoClaw

1

Crea il file src/http-api.ts

Questo file è già stato creato nella cartella del progetto. Contiene un server HTTP minimale sulla porta 3399 che accetta domande e le inietta direttamente nella pipeline di Jarvis.

2

Avvia (o riavvia) NanoClaw

Dopo aver aggiunto il file, riavvia il servizio:

launchctl kickstart -k gui/$(id -u)/com.nanoclaw

Da quel momento il server HTTP sarà disponibile su http://MAC_IP:3399.

3

Testa dal Terminale

Prima di configurare lo shortcut, verifica che funzioni:

curl -X POST http://localhost:3399/ask \ -H "Content-Type: application/json" \ -d '{"text": "ciao Jarvis, come stai?"}'

Dovresti ricevere un JSON con la risposta di Jarvis entro qualche secondo.

📋 Passi Shortcut B — Dettagliati

1

Aggiungi "Dictate Text"

Come nell'Opzione A: cerca Dictate Text, imposta Italian e After Pause.

2

Aggiungi "Get Contents of URL"

Cerca e aggiungi l'azione Get Contents of URL. Configurala così:

  • URL: http://192.168.1.XX:3399/ask (sostituisci con il tuo IP)
  • Method: tocca GET e cambia in POST
  • Headers: tocca Add new header
    Key: Content-Type · Value: application/json
  • Request Body: tocca e seleziona JSON
Configurare il body JSON: dopo aver selezionato JSON come tipo di body, apparirà una sezione con dei campi. Tocca "Add new field" → scegli tipo Text → scrivi come chiave text → nel campo valore tocca il campo e poi il pulsante variabili (in basso a destra della tastiera) → seleziona "Dictated Text".
3

Aggiungi "Get Dictionary Value"

Cerca Get Dictionary Value.
Configura:

  • Get: lascia su Value
  • for Key: scrivi response
  • from: seleziona la magic variable "Contents of URL" (la pillola arancione dell'azione precedente)
4

Aggiungi "Speak Text"

Cerca Speak Text.
Nel campo testo seleziona la magic variable "Dictionary Value" dall'azione precedente.

Opzionale: imposta Language su Italian (Italy) per una voce italiana.

5

Rinomina e aggiungi a Siri

Rinomina lo shortcut in "Jarvis" → ⋮ → Add to Siri → registra la frase.

«Hey Siri, Jarvis»
parla → attendi qualche secondo → Jarvis risponde ad alta voce

💡 Consigli pratici

🔒

Sicurezza del server HTTP

Il server gira solo sulla rete locale e non è accessibile da internet. Se vuoi usarlo fuori casa, aggiungi Cloudflare Tunnel o Tailscale — configurazione avanzata su richiesta.

⌚

Apple Watch

Gli shortcut Siri funzionano anche sull'Apple Watch. Con l'Opzione A, puoi tenere il polso davanti alla bocca e dire "Hey Siri, Jarvis" — il watch apre Telegram (se installato) oppure invia direttamente.

🏠

Widget schermata di blocco

Su iOS 16+ puoi aggiungere lo shortcut come widget nella schermata di blocco: Impostazioni → Schermata di blocco → Personalizza → Widget. Tap sul widget → apre Telegram già pronto per scrivere.
